## ⚙️ Setup & Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/s-leirbag/portfolio.git
   ```
2. Navigate to the project folder:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   pnpm install
   ```
4. Start the development server:
   ```bash
   pnpm dev
   ```
